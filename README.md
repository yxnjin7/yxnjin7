## 📌 주요 프로젝트

### 1. DART 사업보고서 기반 ESG 공시 텍스트 분석

DART 사업보고서에 포함된 ESG 관련 서술을 수집하고, 텍스트 분석을 통해 기업의 ESG 공시 특성을 분석한 프로젝트입니다.
텍스트 전처리, Seed Dictionary 기반 단어 매칭, TF-IDF 점수화, FastText 기반 단어 확장, 회귀분석 등을 활용하여 ESG 공시 텍스트와 평가 등급 간의 관계를 살펴보았습니다.

**주요 키워드**: ESG, DART, 사업보고서, 텍스트 마이닝, TF-IDF, FastText, 회귀분석

---

### 2. 수질 데이터 기반 음용수 판별 모델

수질 측정 데이터를 활용하여 물이 음용 가능한지 여부를 판별하는 머신러닝 분류 모델을 구축한 프로젝트입니다.
결측치 처리, 변수 탐색, 데이터 전처리, 분류 모델 학습 및 성능 평가를 통해 수질 특성이 음용 가능성 판단에 미치는 영향을 분석했습니다.

**주요 키워드**: 수질 데이터, 분류 모델, 머신러닝, 데이터 전처리, 성능 평가

---

### 3. 재무특성을 이용한 기업 부실 예측 모델

기업의 재무 지표를 기반으로 기업 부실 가능성을 예측하는 머신러닝 모델을 구축한 프로젝트입니다.
재무비율 및 기업 특성 변수를 활용하여 부실 기업과 정상 기업을 구분하고, 예측 모델의 성능을 비교·평가했습니다.

**주요 키워드**: 재무 데이터, 기업 부실 예측, 금융 데이터 분석, 머신러닝, 분류 모델
